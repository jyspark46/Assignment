SimpleChat 에 다음 기능을 추가하고 다음 지시에 따라 제출하라.
설계내용은 각 메소드 앞부분에 주석문으로 기술할 것! 
 현재 접속한 사용자 목록 보기 기능
 채팅 문장으로 “/userlist” 를 보내면 현재 접속한 사용자들의 id 및 총 사용자 수를 보여준다. 메소드 : send_userlist()
 자신이 보낸 채팅 문장은 자신에게는 나타나지 않도록 할 것
 금지어 경고 기능
 서버에 금지어 목록을 미리 등록함 (5개 이상)
 채팅 문장에 금지어가 포함되어 있으면 다른 사용자에게 전송하지 않고, 해당 사용자에게만 적절한 경고 메시지를 보낸다.
제출물 : 소스파일과 실행화면캡쳐이미지를 압축한 1개의 파일
변경한 소스파일 (.java) : 소스 내에 반드시 어떠한 전략으로 각 기능이 구현되었는지 설명되어 있어야 하며, 소스 맨 위줄에는 자신의 SimpleChat 저장소의 깃헙 주소가 적혀야 함
각각의 기능이 제대로 구현되었는지를 보여주는 실행화면 캡쳐이미지 (충분히 보여줄 수 있도록 적절한 사용자들이 동시에 채팅하는 장면을 보여야 함)
자신의 github 내 SimpleChat 저장소에 최종 변경된 파일을 올려서 소스가 충돌되고 차이점을 보여주면서 소스버전이 변경되는 장면을 화면캡쳐한 이미지
